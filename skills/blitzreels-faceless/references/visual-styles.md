# Visual Styles Reference

Available visual styles for faceless video generation.

## Realistic
| ID | Name | Description |
|----|------|-------------|
| photography | Photography | Professional DSLR, shallow depth of field, golden hour warmth |
| realism | Realism | Balanced photorealistic with cinematic quality |
| ultra_real | Ultra Real | Hyper-detailed 8K, extreme texture fidelity |

| ID | Name | Description |
|----|------|-------------|
| anime | Anime | Japanese cel animation, vibrant colors, expressive characters |
| shonen | Shonen | Action anime with speed lines, dynamic poses, intense color |
| chibi | Chibi | Super-deformed cute characters with pastel tones |
| manga | Manga | Black & white ink with screentones, high contrast |
| ghibli | Ghibli | Soft watercolor aesthetic inspired by Studio Ghibli |

- **Educational/explainer**: educational, realism, illustration
- **Storytelling/narrative**: cinematic, ghibli, fantasy
